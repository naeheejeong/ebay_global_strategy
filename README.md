# eBay 글로벌 진출 전략 분석

> 이커머스 대규모 데이터 분석 · 785,228건 · 통계 검증

국내 기업의 미국·영국·호주 eBay 진출을 위한 카테고리·마케팅 전략을 실데이터 수집으로 분석한 프로젝트

## 🎯 배경 · 문제정의
- 국내 기업의 글로벌 진출 — 미국·영국·호주에서 실제 진입 가능한 카테고리·전략 도출
- 기존 리서치로는 실시간 판매·판매자 행동 파악 불가 → **실데이터 직접 수집이 필수**
- 핵심 과제: 마케팅(할인) 적용 여부의 효과 비교

## 🗂 데이터 · 방법
- eBay Browse API **106회** 호출(US·GB·AU) + Selenium 동적 크롤링 → 약 100만 건 수집
- 크롤링 콤마 버그 발견 → 78만 건 전량 폐기, 코드 수정 후 재수집 (데이터 정합성 확보)
- 정제 후 최종 분석 데이터셋 **785,228건**, 파생변수 18개 설계 (RFM·Churn·판매자 유형)
- 국가·카테고리·규모 층화 후 t-test로 유의성 검증

## 📈 결과 · 임팩트
- 마케팅 적용 판매자 1인당 평균 매출 **4배↑** — t = 4.96, p ≈ 7.08×10⁻⁷ (통계적으로 유의)
- 국가별 진출 전략 도출: US(B2B 대형품목·고정가+무료배송), GB(프리미엄·리퍼, 1인당 매출 US 대비 1.9배), AU(니치 퍼스널케어·Best Offer)
- Tableau 인터랙티브 대시보드 4종 + RFM 기반 잠재 전환·이탈 판매자 식별
